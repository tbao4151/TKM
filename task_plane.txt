TASK 4: Kiem tra MPLS kernel native

Bat buoc:
- sudo modprobe mpls_router
- sudo modprobe mpls_iptunnel
- sysctl net.mpls.platform_labels
- Bat net.mpls.conf.<interface>.input=1 tren interface PE/P.
- Cau hinh route MPLS that bang:
  - ip route add <prefix> encap mpls <label> via inet <next-hop> dev <interface>
  - ip -f mpls route add <label> as <new-label> via inet <next-hop> dev <interface>
  - ip -f mpls route add <label> via inet <next-hop> dev <interface>
- Dung tcpdump tren core link de bat goi co ethertype MPLS.
- P router khong duoc chi dua vao static IP route toi LAN dich trong mode MPLS; P router phai forward bang `ip -f mpls route`.

Can tao/cap nhat:
- code/network_config.py
- code/topology_mininet.py
- code/performance_test.py
- docs/mpls_kernel_setup.md neu can giai thich rieng
- results/mpls_routes.txt
- results/tcpdump_mpls.txt

Tieu chi dat:
- Ping lien chi nhanh chay trong `--mode mpls`.
- `results/mpls_routes.txt` co route `ip -f mpls route show` tren PE/P.
- `results/tcpdump_mpls.txt` co goi MPLS that bat tren core link.
